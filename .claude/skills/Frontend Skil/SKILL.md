# Frontend Skill: Components, Pages, Layouts, and Styling

## Instructions

1. **Component building**
   - Create reusable UI components
   - Use semantic HTML elements
   - Follow accessibility best practices

2. **Page structure**
   - Define clear sections (header, main, footer)
   - Organize content logically
   - Ensure responsive design

3. **Layouts**
   - Use CSS Flexbox and Grid for layout
   - Support mobile-first and desktop views
   - Maintain consistent spacing and alignment

4. **Styling**
   - Apply modern CSS techniques (variables, custom properties)
   - Use consistent color schemes and typography
   - Include hover/focus states for interactivity

## Best Practices
- Write clean, maintainable code
- Prioritize performance and accessibility
- Use modular CSS or CSS-in-JS when appropriate
- Test components across different screen sizes and browsers

## Example Structure
```html
<section class="page-layout">
  <header class="header">
    <nav class="nav">Navigation</nav>
  </header>
  <main class="main-content">
    <article class="component-card">
      <h2>Component Title</h2>
      <p>Component content goes here.</p>
      <button class="action-button">Click Me</button>
    </article>
  </main>
  <footer class="footer">Footer content</footer>
</section>
